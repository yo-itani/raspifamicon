# APU (Audio Processing Unit)

ref: https://www.nesdev.org/wiki/APU

## 概要

CPU (2A03) 内蔵。5チャンネル。フレームカウンタ (~240Hz) がエンベロープ・スイープ・長さカウンタを駆動。

## チャンネル

| チャンネル | レジスタ | 説明 |
|-----------|---------|------|
| Pulse 1 | $4000-$4003 | 矩形波、デューティ比4種、エンベロープ、スイープ |
| Pulse 2 | $4004-$4007 | Pulse 1と同じ |
| Triangle | $4008-$400B | 三角波 (32段階)、音量制御なし、リニアカウンタ |
| Noise | $400C-$400F | LFSR疑似乱数ノイズ、2モード (long/short)、エンベロープ |
| DMC | $4010-$4013 | 1bitデルタ変調サンプル再生 (7bit PCM出力)、$C000-$FFF1から読込 |

## 制御レジスタ

- $4015 (W): チャンネル有効/無効
- $4015 (R): 長さカウンタ状態、DMC/フレームIRQフラグ
- $4017: フレームカウンタモード (4/5ステップ)、IRQ禁止
