# コントローラ入力

ref: https://www.nesdev.org/wiki/Standard_controller

## レジスタ

- $4016: コントローラ1 (R/W)
- $4017: コントローラ2 (Rのみ、Wはフレームカウンタ)

## 読み取りプロトコル

1. $4016に$01を書込 (ストローブ: ボタン状態をラッチ)
2. $4016に$00を書込 (シリアルモードに切替)
3. $4016 (または$4017) を8回読込。各読込でbit0に1ボタン分

## ボタン順序 (読込順)

A → B → Select → Start → Up → Down → Left → Right

## 注意: DMC競合

DMCチャンネル動作中、DMAがコントローラシフトレジスタに余分なクロックを送り、
読み取りが壊れることがある (幻の右押し)。
対策: 複数回読んで比較、またはOAM DMAとタイミングを同期。
