# PPU (Picture Processing Unit)

ref: https://www.nesdev.org/wiki/PPU

## 概要

256x240ピクセル出力、262スキャンライン/フレーム (NTSC)。
3 PPUサイクル = 1 CPUサイクル。
VBlank中 (~20スキャンライン, ~2270 CPUサイクル) にVRAM更新可能。

## PPUアドレス空間

```
$0000-$1FFF  パターンテーブル (8KB, CHR-ROM/RAM、タイルグラフィック)
             $0000-$0FFF  パターンテーブル0 (256タイル, 8x8)
             $1000-$1FFF  パターンテーブル1 (256タイル, 8x8)
$2000-$2FFF  ネームテーブル (2KB VRAM)
             各テーブル = 32x30タイル(960B) + 属性テーブル(64B)
$3F00-$3F1F  パレットRAM (32バイト)
             背景4パレット + スプライト4パレット (各4色)
```

## OAM (Object Attribute Memory)

256バイト = 64スプライト x 4バイト
- byte 0: Y座標
- byte 1: タイルインデックス
- byte 2: 属性 (反転、優先度、パレット)
- byte 3: X座標

制約: 1スキャンラインに最大8スプライト

## CPU側PPUレジスタ ($2000-$2007)

| アドレス | 名前 | 機能 |
|---------|------|------|
| $2000 | PPUCTRL | NMI有効、スプライトサイズ、パターンテーブル選択、VRAM増分 |
| $2001 | PPUMASK | 色強調、スプライト/背景表示有効、左端クリッピング |
| $2002 | PPUSTATUS | VBlankフラグ、スプライト0ヒット、スプライトオーバーフロー |
| $2003 | OAMADDR | OAMアドレス |
| $2004 | OAMDATA | OAMデータR/W |
| $2005 | PPUSCROLL | スクロール位置 (2回書込: X→Y) |
| $2006 | PPUADDR | VRAMアドレス (2回書込: 上位→下位) |
| $2007 | PPUDATA | VRAMデータR/W |
